# Kioskeo — App POS para Kioscos

Tu punto de venta inteligente. Escanea, cobra y controla tu stock desde el celular.

## Funciones

- Escaneo de productos por codigo de barras
- Control de stock con alertas visuales
- Cobro rapido con calculo de vuelto
- Metodos de pago: efectivo, transferencia, cuenta corriente (fiado)
- Gestion de clientes con cuenta corriente
- Importacion masiva de productos (PDF/Excel/CSV)
- Accesos rapidos a productos favoritos
- Backup y restauracion de datos
- Tema oscuro y claro
